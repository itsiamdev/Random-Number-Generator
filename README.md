# Random Number Generator 🎲

A simple and intuitive web-based random number generator that lets you generate random numbers within a specified range. Perfect for games, simulations, or any situation where you need a quick random number! 🎯

## Features ✨
- Set custom minimum and maximum values 🔢
- Instant generation with a single click ⚡
- Clean and responsive design 📱
- Works directly in your browser 🌐

## Live Demo 🚀
Check out the live version here: [https://iam269.github.io/Random-Number-Generator/](https://iam269.github.io/Random-Number-Generator/)

## How to Use 📖
1. Enter your desired minimum value in the "Min" field
2. Enter your desired maximum value in the "Max" field
3. Click the "Generate" button
4. Your random number will appear below! 🎉

Enjoy generating random numbers! 🎲✨
